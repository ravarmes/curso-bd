Práticas 4.3 — DML: Inclusão, Exclusão e Alteração

Comandos DML

INSERT INTO tabela (col1, col2)
VALUES ('valor1', 'valor2');

UPDATE tabela
SET col1 = 'novo_valor'
WHERE id = 1;

DELETE FROM tabela
WHERE id = 1;

⚠️ CUIDADO: Sem WHERE, UPDATE e DELETE afetam TODAS as linhas!
