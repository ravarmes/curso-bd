Práticas 4.1 — Introdução à SQL e seus Subconjuntos

Os 5 Subconjuntos da SQL

DDL

Data Definition Language

CREATE, ALTER, DROP — define estruturas

DML

Data Manipulation Language

INSERT, UPDATE, DELETE — manipula dados

DQL

Data Query Language

SELECT — consulta dados

DCL

Data Control Language

GRANT, REVOKE — controle de acesso

DTL

Data Transaction Language

COMMIT, ROLLBACK, SAVEPOINT — transações
