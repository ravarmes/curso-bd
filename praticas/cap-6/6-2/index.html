Práticas 6.2 — Índices: Criação, Tipos e Desempenho

Tipos de Índices

B-Tree

Padrão. Ideal para =, <,>, BETWEEN

CREATE INDEX idx
ON tabela(col);

Hash

Apenas igualdade (=)

CREATE INDEX idx
ON tabela
USING HASH(col);

GIN

Arrays, JSONB, Full-text

CREATE INDEX idx
ON tabela
USING GIN(col);

Índice Composto

CREATE INDEX idx_cat_preco
ON produto(categoria_id, preco);

A ordem importa! O índice favorece consultas que filtram na mesma ordem.

✅ Usa o índice:

  • WHERE categoria_id = 1
  • WHERE categoria_id = 1 AND preco > 100

❌ NÃO usa o índice:

  • WHERE preco > 100 (sem categoria_id)
