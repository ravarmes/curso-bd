Tratando Valores Nulos: fillna()

Antes (com NULL):

nome idade nota

3 valores nulos detectados

Depois (fillna):

nome idade nota 
# Opções de tratamento
df['idade'].fillna(df['idade'].mean(), inplace=True)  # média
df['nota'].fillna(0, inplace=True)                    # valor fixo
df.dropna()                                            # remover linhas

One-Hot Encoding

Transforma variáveis categóricas em colunas binárias (0/1).

Antes:

id categoria
1 Ficção
2 Técnico
3 Ficção
4 Infantil

Depois (One-Hot):

id cat_Ficção cat_Técnico cat_Infantil 
df = pd.get_dummies(df, columns=['categoria'], prefix='cat')

Normalização / Scaling

Min-Max Scaling

Escala para [0, 1]

X' = (X - min) / (max - min)
Original Normalizado

Z-Score (Standardization)

Média=0, Desvio=1

Z = (X - μ) / σ
Original Z-Score

Pipeline de Pré-processamento

  1. Verificar duplicatas: df.drop_duplicates()
  2. Tratar nulos: df.fillna() ou df.dropna()
  3. Tratar outliers: IQR, clip()
  4. Codificar categóricas: pd.get_dummies() (One-Hot)
  5. Normalizar numéricas: MinMaxScaler ou StandardScaler
from sklearn.preprocessing import MinMaxScaler, StandardScaler

scaler = MinMaxScaler()
df['preco_norm'] = scaler.fit_transform(df[['preco']])
